# Project Management Tool

A full-stack Project Management Tool built using HTML, CSS, JavaScript, Node.js, Express.js and MongoDB.

## Features
- User Registration & Login
- Create Projects
- Create Tasks
- Assign Tasks
- Update Task Status
- Add Comments
- Delete Tasks
- Delete Comments

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB
