# 🚀 TaskFlow - Project Management Tool

A full-stack project management platform that helps users create projects, manage tasks, track progress, and collaborate through comments.

Built using **Node.js, Express.js, MongoDB, HTML, CSS, and JavaScript**.

---

## 📌 Features

### 👤 User Management 
- User registration 
- User login authentication 
- Secure user data storage using MongoDB
- 
### 📁 Project Management
- Create new projects
- Store project details permanently in MongoDB
- View all created projects
- Track project creator and creation date

### ✅ Task Management
- Create tasks under projects
- View all tasks
- Update task status
- Delete tasks
- Add comments on tasks

### 💬 Comment System
- Add comments to tasks
- View project/task discussions

### ⚡ Backend API
- RESTful API architecture
- Express.js based server
- MongoDB database integration
